## [1.0.0] - 2024-06-25

### Added
- Initial release of DookAI
- Google Sign-In authentication with Firebase
- Auto-login functionality for returning users
- Session persistence and secure logout
- Beautiful splash screen with smooth animations
- Modern login screen with Google Sign-In button
- Personalized home screen with welcome message
- New Chat button for starting conversations
- Chat list with pinned chats section
- Search functionality for finding chats
- Real-time chat interface with AI responses
- Message streaming with typing animation
- Copy message functionality
- Favorite/bookmark messages
- Regenerate response option
- Rename and delete chats
- User profile page with account information
- Settings page with dark mode toggle
- Light and dark theme support
- Firebase Firestore integration for data persistence
- Cloud storage for user data
- Markdown formatting with code syntax highlighting
- Responsive design for all screen sizes
- Optimized performance for mobile devices
- Secure Firebase security rules
- Error handling and user-friendly messages
- Loading states and animations
- Empty state screens
- Suggested prompts for user guidance
